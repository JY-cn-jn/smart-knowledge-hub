# Python Flask 基础入门

Flask 是一个轻量级的 Python Web 框架。

它适合用来开发：

- 个人项目
- 小型网站
- API 服务
- 数据展示页面

在这个智能知识库项目中，Flask 主要负责：

1. 显示首页
2. 显示文章列表
3. 显示文章详情页
4. 接收搜索请求
5. 从数据库读取文章信息

Flask 的优点是结构简单，容易理解，也很适合初学者一步一步学习。
