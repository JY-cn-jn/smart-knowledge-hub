# 相关文章推荐功能的设计

相关文章推荐是智能知识库系统的核心功能之一。

当用户阅读一篇文章时，系统可以在文章底部推荐相关内容。

第一版可以使用简单规则：

1. 分类相同，加分
2. 标签重合，加分
3. 标题关键词相似，加分
4. 摘要内容相似，加分

例如，用户正在阅读 Flask 文章，系统可以推荐：

- Python Flask 基础入门
- Flask 读取 Markdown 文件
- MySQL 在知识库系统中的作用

后期可以升级成语义推荐。

语义推荐可以通过 embedding 计算文章之间的相似度。
