# RSS 自动收集文章的设计思路

RSS 是一种常见的信息订阅格式。

很多博客、新闻网站和技术网站都会提供 RSS 源。

未来这个系统可以定时读取 RSS 源，然后自动完成：

1. 检查有没有新文章
2. 获取文章标题和链接
3. 下载文章正文
4. 保存成 Markdown 文件
5. 写入 MySQL 数据库
6. 在网页中显示新文章

这样知识库就可以从“手动添加文章”升级成“自动收集文章”。

这是这个项目未来非常重要的功能。
